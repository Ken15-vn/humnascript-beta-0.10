# 🌍 HumanScript Beta 0.10

## 🔥 Update Beta 0.10 !!!
This version builds on the stable foundation of 0.05 and introduces powerful new calculation features:

* **New `set` command**: Create variables to store numeric values (e.g., `set "score" 100`).
* **New `add` command**: Perform addition directly on your stored variables.
* **Enhanced `write` command**: Now supports printing variable values by calling their names.
* **Inherited Features**: Includes all stable features from 0.05 (`color`, `clear`, `wait`, `write`).

---

**"Made for Humans!"**
HumanScript is a lightweight programming language. Goal: Make coding as easy as writing English. No `;` or `{}`.

---

## 📜 Current Commands:

* **set "name" value**: Create a variable and assign it a number.
* **add "name" value**: Add a number to an existing variable.
* **write "message" or "variable"**: Output text or variable values to the console.
* **color "name"**: Change text color (green, red, yellow, blue, cyan, white).
* **clear**: Clear the entire console window.
* **wait X**: Pause program execution for X seconds.

## 🚀 How to Use (4 Steps)

1.  **Download**: Download the `humanscript beta 0.10.zip` file and extract it to your folder.
2.  **Setup**: Make sure you have **Python 3.12** or higher installed on your system.
3.  **Code**: Open `ide.human` with any text editor and write the commands you want to execute.
4.  **Run**: Double-click **`run_for_others.bat`** to see your code in action.

## 📜 File Structure

* **humanscript_beta_0.10.py**: The core interpreter containing the new math logic.
* **ide.human**: Your script file for testing the new Beta 0.10 features.
* **run_for_others.bat**: Quick-launch file for general users.
* **run.bat**: Template for manual path configuration (Used on developer's PC).
* **Error correction instructions.txt**: Help guide for Python path configuration.
* **humanscript beta 0.10.zip**: The complete package for easy sharing.
